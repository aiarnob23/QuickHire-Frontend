# 🚀 QuickHire – Modern Job Board Application

**QuickHire** is a high-performance, mini job board platform built with **Next.js**. It provides a seamless experience for job seekers to find opportunities and a streamlined interface for admins to manage listings.

---

## ✨ Features

### 🧑‍💻 For Candidates
- **Dynamic Job Discovery:** Browse jobs in a responsive, high-performance grid layout.
- **Advanced Filtering:** Instant search by keywords or filter by **Category** and **Location**.
- **Detailed Insights:** Dedicated pages for full job descriptions and company info.
- **Streamlined Application:** Built-in form with validation for Name, Email, and Resume URL.

### 🛠 For Administrators
- **Management Dashboard:** A clean interface to oversee all active listings.
- **CRUD Capability:** Easily **Create** new job postings or **Delete** outdated ones.

### 🎨 Design & UX
- **Figma-to-Code Precision:** Developed to match professional design specifications.
- **Mobile-First:** Fully responsive UI powered by **Tailwind CSS**.
- **Modern Components:** Accessible UI elements via **Shadcn/ui**.

---

| Feature | Technology |
